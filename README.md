# Music Studio — Download · Identify · Complete

## Aperçu
Notebook Google Colab tout-en-un pour télécharger l'audio d'un Reel Instagram ou d'un Short YouTube, identifier le genre musical via plusieurs modèles HuggingFace, et compléter automatiquement l'extrait en un son de 3min30 grâce à MusicGen (Meta).

## Fonctionnalités
- Téléchargement audio depuis Instagram Reels & YouTube Shorts (MP3, WAV, MP4)
- Identification du style musical via 3 modèles HuggingFace en parallèle (CLAP, AST, Wav2Vec2)
- Génération des parties manquantes (intro + outro) par chunks de 30s pour atteindre 3min30
- Interface graphique unifiée dans une seule cellule Colab

## Technologies
- Python
- Google Colab
- yt-dlp
- HuggingFace Transformers (MusicGen Medium, CLAP, AST)
- ipywidgets
